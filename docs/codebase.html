API ROUTES

8 routes
src/app/api/campaigns/create/route.ts
POST /api/campaigns/create. Authenticates user, validates URL, inserts campaign row (status=generating), triggers server-to-server call to /process (non-blocking, forwards cookies), returns { campaignId }.
supabase/server campaigns INSERT
src/app/api/campaigns/[campaignId]/process/route.ts
POST /api/campaigns/[id]/process. Heavy pipeline (maxDuration=300s): (1) scrapeUrl via Firecrawl, (2) buildBrandProfile via Claude, (3) findCompetitorAds via Meta Ad Library (non-fatal), (4) storeBrandAssets to Supabase Storage, (5) save profile/assets/copy to campaigns, (6) generateCampaignAds (2 UGC videos). On failure at any step, sets status=draft. Langfuse tracing throughout.
scraper.ts brand-profiler.ts competitor-research.ts ad-generator.ts asset-storage.ts langfuse.ts supabase/server (admin) Firecrawl Anthropic Meta Ad Library Fal.ai Supabase Storage Langfuse campaigns SELECT/UPDATE ads INSERT
src/app/api/campaigns/[campaignId]/status/route.ts
GET /api/campaigns/[id]/status. Returns campaign status, brandProfile, adCopy, brandAssets, and all ads. Used for polling during generation (every 3s from onboard page).
supabase/server campaigns SELECT ads SELECT
src/app/api/campaigns/[campaignId]/ads/route.ts
GET /api/campaigns/[id]/ads. Returns ads for campaign. Optional ?preview=true filter for preview-only ads.
supabase/server ads SELECT
src/app/api/campaigns/[campaignId]/checkout/route.ts
POST /api/campaigns/[id]/checkout. Validates budget (min $5 / 500 cents). Creates Stripe Checkout Session (one-time payment, campaign metadata). Updates campaign: daily_budget, stripe_payment_intent_id, status=payment_pending. Returns { checkoutUrl }.
supabase/server (service) Stripe campaigns SELECT/UPDATE
src/app/api/campaigns/[campaignId]/go-live/route.ts
POST /api/campaigns/[id]/go-live. Requires status=ready. Creates Meta campaign (OUTCOME_TRAFFIC, daily budget with 10% commission, converted to INR), ad set (Advantage+ US targeting), and ads (video or image creatives). Updates campaign + ad rows with Meta IDs, sets status=live. maxDuration=120s.
meta-ads.ts supabase/server (service) Meta Marketing API campaigns SELECT/UPDATE ads SELECT/UPDATE
src/app/api/campaigns/[campaignId]/pause/route.ts
POST /api/campaigns/[id]/pause. Accepts { action: "pause" | "resume" }. Calls meta-ads.ts pauseCampaign() or resumeCampaign(). Updates campaign status to "paused" or "live".
meta-ads.ts supabase/server (service) Meta Marketing API campaigns SELECT/UPDATE
src/app/api/webhooks/stripe/route.ts
POST /api/webhooks/stripe. Stripe webhook handler. Verifies signature with STRIPE_WEBHOOK_SECRET. Handles checkout.session.completed: extracts campaign_id from metadata, updates campaign status to "ready" using admin client.
supabase/server (admin) Stripe Webhooks campaigns UPDATE
src/app/api/cron/sync-metrics/route.ts
GET /api/cron/sync-metrics. Hourly cron (vercel.json). Auth via Bearer CRON_SECRET. Fetches all live campaigns with meta_campaign_id. For each, calls Meta Graph API for ad insights (impressions, clicks, ctr, cpc, spend). Matches ads by meta_ad_id, inserts metrics rows.
supabase/server (admin) Meta Graph API v19.0 campaigns SELECT ads SELECT metrics INSERT
src/app/api/admin/library/route.ts
GET + POST /api/admin/library. Admin-only (checks ADMIN_EMAIL env var). GET: lists all ad_library templates. POST: accepts FormData (category, prompt, notes, optional visual file), uploads visual to Supabase Storage "ad-library" bucket, inserts row.
supabase/server (service) Supabase Storage ad_library SELECT/INSERT

LIB (UTILITIES)

10 files
src/lib/scraper.ts
Website scraper using Firecrawl. Extracts markdown content + structured JSON (logo, favicon, product images, brand colors, OG image, title, description, target audience, key features). Returns ScrapeResult with BrandAssets. Langfuse-traced.
langfuse.ts Firecrawl Langfuse
src/lib/brand-profiler.ts
Two functions: (1) buildBrandProfile() sends scrape data to Claude claude-sonnet-4-6, returns BrandProfile (category, tone, audience, value props, product name) + AdCopy (headline <25 chars, primaryText <100 chars, description <25 chars). (2) generateVideoScripts() generates 2 UGC ad scripts with different hooks (curiosity/FOMO) and random settings (city, cafe, park, etc), 40-50 words each. Expert UGC copywriter system prompt.
langfuse.ts Anthropic (claude-sonnet-4-6) Langfuse
src/lib/competitor-research.ts
Searches Meta Ad Library API for competitor ads. Uses top 3 value props as search terms, US-targeted, limit 5 results. Returns array of CompetitorAd (id, page_name, ad_creative_body, ad_snapshot_url). Non-fatal: returns empty array on error.
langfuse.ts Meta Ad Library API Langfuse
src/lib/fal-client.ts
Fal.ai client with 3 capabilities: (1) generateImageAd() using nano-banana-2 (base or edit model with reference images, fallback to base). (2) generateVideoAd() pipeline: generates raw video (model from ad_config, default sora-2), then optionally adds subtitles. (3) addSubtitles() using auto-subtitle model (Montserrat bold, orange highlights, 3 words/subtitle). Aspect ratios: 1:1, 9:16, 16:9, 4:5.
langfuse.ts ad-config.ts Fal.ai Langfuse
src/lib/ad-generator.ts
Orchestrates ad generation for a campaign. Generates 2 UGC video scripts (via brand-profiler.ts), builds video prompts (12s UGC-style, 9:16, iPhone look), generates videos sequentially via fal-client.ts (each saved to DB immediately for partial progress visibility). Updates campaign status to "ready" when done. Throws if all generations fail.
fal-client.ts brand-profiler.ts supabase/server (admin) langfuse.ts ads INSERT campaigns UPDATE
src/lib/meta-ads.ts
Meta Marketing API wrapper (Graph API v19.0). Functions: createCampaignWithBudget() (OUTCOME_TRAFFIC), createAdSet() (Advantage+ US, LOWEST_COST_WITHOUT_CAP), uploadAdImage() (base64), createImageAdCreative(), createVideoAdCreative() (upload video, poll for thumbnail, fallback to brand image), createAd(), pauseCampaign(), resumeCampaign(). All use metaFetch() helper with error parsing.
Meta Marketing API v19.0
src/lib/asset-storage.ts
Downloads external brand images (logo, favicon, OG, product images) and re-uploads to Supabase Storage "ad-creatives" bucket. Filters: skips files <1KB (tracking pixels) or >10MB. Parallel uploads. Returns new BrandAssets with Supabase public URLs.
supabase/server (admin) Supabase Storage
src/lib/ad-config.ts
Fetches ad generation config from ad_config table with 1-minute cache. Keys: video_model (default: fal-ai/sora-2/text-to-video), video_duration (4), video_resolution (720p), video_aspect_ratio (9:16), subtitle_enabled (true). Falls back to hardcoded defaults on error.
supabase/server (admin) ad_config SELECT
src/lib/langfuse.ts
Initializes singleton Langfuse client for LLM observability. Configured via LANGFUSE_SECRET_KEY, LANGFUSE_PUBLIC_KEY, LANGFUSE_BASEURL (default: us.cloud.langfuse.com). Exported as `langfuse`.
Langfuse
src/lib/supabase/server.ts
Three Supabase client factories: (1) createClient() - cookie-based server client with anon key (for authenticated routes), (2) createServiceClient() - cookie-based with service role key (for admin operations in request context), (3) createAdminClient() - no cookies, direct service role (for webhooks, cron, background jobs). All typed with Database.
Supabase
src/lib/supabase/client.ts
Browser-side Supabase client using createBrowserClient with anon key. Used by client components (landing page, login, signup) for auth operations.
Supabase
